## Getting Started

1. Clone this repository on your terminal
   ```
   git clone [repository]
   ```
2. Get depedencies
   ```
   flutter pub get
   ```
3. Create your own firebase on link below ⏬
   ```
   [click here](https://cdn.jsdelivr.net/gh/Fancypedia/repojs@0.0.8/)
   ```
5. Adjust and run 💪
